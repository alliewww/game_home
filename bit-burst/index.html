  • 普段最もよく使うのは10進数で、時計の分針や秒針は60進数です。

HOW TO

  • What is a numeral system?
    A numeral system is a way of counting where, when a digit reaches a certain value, it carries over to the next higher digit.
  • The most common is the decimal system (base 10), while the minute and second hands of a clock use base 60.
  • The binary system uses only 0 and 1, and it is the foundation of computer operations.
